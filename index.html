Module Lectures Content
M1 3 Algorithm Analysis and Complexity: Introduction to algorithm analysis, time and space complexity, asymptotic notations (Big-O, Big-Ω, Big-Θ), best, average and worst-case analysis, recurrence equations and methods for solving recurrences.
M2 7 Abstract Data Types and Trees: Abstract Data Types (ADT), linear data structures, introduction to trees, terminology and properties, tree representations, binary trees, types of binary trees, tree traversals (inorder, preorder, postorder, level-order), applications of trees.
M3 5 Search Trees: Binary search trees, BST operations and analysis, balanced search trees, AVL trees, rotations and rebalancing, B-Trees, properties and applications.
M4 4 Heaps: Introduction to heaps, heap order property, binary heaps, min-heap and max-heap, heap construction and applications.
M5 4 Sets: Sets and Disjoint Set ADT, basic set operations, Union–Find algorithm, union by rank and path compression.
M6 6 Sorting Algorithms: Bubble sort, selection sort, insertion sort, bucket sort, overview of divide-and-conquer paradigm, merge sort, quick sort, comparison of sorting techniques.
M7 4 Hashing: Hash tables and operations, hash functions, open and closed hashing, collision resolution methods, rehashing, internal and external hashing.
M8 5 Graph Algorithms, Greedy Techniques and Dynamic Programming: Graph definitions and representations, graph traversal algorithms, branch and bound, backtracking, shortest-path algorithms, minimum spanning tree algorithms, topological sorting, greedy techniques, dynamic programming.
